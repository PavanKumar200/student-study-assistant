# Student Study Assistant

An AI-powered search tool for students that provides:
- AI-generated answers using Google Gemini
- Top YouTube educational videos
- Related articles
- Study materials (PDFs/PPTs)
- Cheat sheets and quick references

## Technologies Used
- HTML, CSS, JavaScript
- Google Gemini API
- YouTube Data API
- Google Custom Search API
